HTML Table Reference

Basic Table Structures

Tag Name Description
<table> Table Element Used to create a table.
<tr> Table Row Used to create a table row.
<td> Table Data The data of each cell within the table.

Table Headings

Tag Name Description
<th> Table Heading Similar to the <th> element, but to represent a heading for a column or row
<tr> Table Row Used to create a table row.
<td> Table Data The data of each cell within the table.

Spanning

Tag Name Description
<colspan> Column Span For when a cell needs to stretch across mulitlple cells.
<rowspan> Row Span For when a cell needs to stretch across mulitlple cells.

Long Tables

Tag Name Description
<thead> Table Heading Section The heading of the table should sit within <thead>
<tbody> Table Body Section The body of the table should sit within <tbody>
<tfoot> Table Footer Section The footer of the table should sit within <tfoot>
